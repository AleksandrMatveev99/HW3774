# Создание веток 

* Ускоренный(комбо команд) вариант : git checkout -b branchName

1. git branch branchName
2. git checkout branchName

## Команды для работы с ветками

**git branch** показывает какие есть ветки
**git branch branchName** создаем ветку
**git checkout branchName** переход в ветку
**git merge branchName** слияние веток

## При слиянии 
* делать коммит не надо (если не было конфликтов)
* делаем коммит, когда решили конфликт (возникает, когда изменения в одних и тех же строках)

Команда для вызова списка сохраненных ихменений "git log"
Команда для вызова списка в виде дерева/графа веток "git log --graph"
